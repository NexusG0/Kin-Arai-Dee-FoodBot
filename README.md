## How to run it on your own machine

   1. Setup Environment
   
      For Windows
      
      ```
      $ python3 venv .venv
      $ source .venv/bin/activate
      ```
      
      For Linux/macOS
   
      ```
      $ python3 -m venv venv
      $ source venv/bin/activate
      ```

   3. Install the requirements

      ```
      $ pip install -r requirements.txt
      ```

   4. Run the app

      ```
      $ streamlit run app.py
      ```
